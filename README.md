# 🗒️ AgentAI ScholarFI
ScholarFi which is an AI-powered personal financial coach pursuing the purpose to help international students in London make challenging financial and academic decisions.

## ✨ Features
-AI-powered Financial mentor 🖥️<br/> 
-Budgeting and Expense Insights 🆕<br/> 
-Data-driven recommendations ✅<br/> 
-Ethical and responsible AI Design 🔍 <br/> 
-Full-stack Architecture ✅<br/> 
-Scalable and Extensible system design 🖥️<br/> 
-Testing and validation 📱<br/> 

## 🧠 Systems Used
- Visual Studio Code

## 🧩 Resources
- OpenAI GPT API <br/> 
- Python and PostgreSQL (Backend) <br/> 
- Streamlit and Altair (Frontend) <br/> 

## 🚀 Deployment (Link)
https://scholarfi.streamlit.app/
